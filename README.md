PRINCE AGENCY

Project Name:
Prince Agency

Description:
Prince Agency is a modern web project built with Next.js, React, and Tailwind CSS.
This project is designed for fast performance, clean UI, and scalable frontend development.

Tech Stack:
- Next.js 16.1.6
- React 19.2.3
- React DOM 19.2.3
- Tailwind CSS v4
- ESLint 9

Project Scripts:
- npm run dev    : Start development server
- npm run build  : Create production build
- npm run start  : Run production server
- npm run lint   : Check code quality with ESLint

Getting Started:
1. Install dependencies
   npm install

2. Run the development server
   npm run dev

3. Open in browser
   http://localhost:3000

Production:
1. Build the app
   npm run build

2. Start production server
   npm run start

Folder Note:
Main app source is inside the project folder:
prince-agency/
